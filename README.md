# opencode-project

Repositorio redundante para proyectos editables desde OpenCode.

## Finanzas familiares

La aplicacion web de finanzas familiares esta en `finanzas-familiares/`.

Para ejecutarla:

```powershell
cd finanzas-familiares
python app.py
```

Tambien incluye `Dockerfile` y `docker-compose.yml` para despliegue en VPS.
